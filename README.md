## üí° Project Description

**DotnetHotelManagementBE** is a robust backend system built using **ASP.NET Core Web API**. It is designed to provide all the necessary **RESTful APIs** required for the comprehensive management of a hotel system.

The system handles core data management tasks, including:
* **Room Management:** Managing room details, types, and status (vacant, cleaning, occupied).
* **Booking Management:** Creating, retrieving, updating, and canceling customer reservations.
* **Customer Management:** Storing and managing customer information.
* **Authentication & Authorization:** Securing API endpoints using **JWT**.

---

## ‚ú® Key Features

* **RESTful APIs:** Provides standardized REST endpoints for easy integration with any Frontend application (React, Angular, Vue, Mobile).
* **JWT Security:** Implements token-based security using **JSON Web Tokens** for user authentication.
* **Entity Framework Core (Code First):** Utilizes EF Core with a Code First approach for database migration and management.
* **Swagger/OpenAPI:** Automatically generates API documentation, allowing for easy testing and integration via a user-friendly UI.
* **Pagination, Sorting & Filtering:** Supports advanced querying for efficient handling of large datasets.
